== Logging
`ConnectivityBluetooth` allows injecting a logging interface:

[tabs]
====
Kotlin::
+
--
[source,kotlin]
----
Logger.addLogDelegate(myCustomLogDelegate)
----
--
Swift::
+
--
[source,swift]
----
Logger.Companion().addLogDelegate(delegate: myCustomLogDelegate)
----
--
